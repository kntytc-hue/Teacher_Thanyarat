# 📘 ระบบแอปพลิเคชันจัดการเรียนการสอนออนไลน์ (Static Web Application)
## รายวิชา 31909-1001 ระบบปฏิบัติการ (Operating Systems)
**ระดับประกาศนียบัตรวิชาชีพชั้นสูง (ปวส.) แผนกวิชาเทคโนโลยีคอมพิวเตอร์**
*พัฒนาและออกแบบสถาปัตยกรรมโดย: อาจารย์ธัญญรัตน์ ตาเล๊ะ*

---

## 💻 ภาพรวมเทคโนโลยีที่เลือกใช้ (Tech Stack)
ระบบนี้ถูกพัฒนาขึ้นในรูปแบบ **Static Web Application 100%** เพื่อให้สามารถทำงานได้รวดเร็ว ปลอดภัย และไม่จำเป็นต้องเชื่อมต่อกับ Backend Database Server หรือติดตั้ง Node.js/Framework ใดๆ ให้ยุ่งยาก รองรับการเปิดใช้งานโดยตรงจากเว็บเบราว์เซอร์หรืออัปโหลดขึ้น Web Server ทั่วไปได้ทันที

* **โครงสร้าง (Structure):** HTML5 (Semantic Tags)
* **การจัดสไตล์ (Styling):** CSS3, Bootstrap v5.3 (Material & Glassmorphism Design)
* **ระบบโต้ตอบ (Interactivity):** JavaScript ES6 แกนหลักแบบแยกไฟล์
* **แอนิเมชัน (Animation):** AOS Animation Engine, Animate.css
* **ระบบฐานข้อมูลฝั่งไคลเอนต์ (Storage):** HTML5 LocalStorage Web Storage API
* **ส่วนประกอบเสริม (UI Elements):** Chart.js (วาดกราฟสถิติ), SweetAlert2 (หน้าต่างแจ้งเตือนอัจฉริยะ)

```text

├── index.html                  # หน้าแรกของบทเรียนและข้อมูลภาพรวมรายวิชา
├── dashboard.html              # หน้ากระดานวิเคราะห์สถิติคะแนนและการออกเกียรติบัตร
├── quiz.html                   # หน้าแกนควบคุมระบบข้อสอบ (ถ้ามี)
├── pages/
│   ├── unit-01.html            # หน้าเนื้อหาเชิงลึกและระบบควิซหน่วยที่ 1
│   └── unit-02.html ถึง 15.html # หน้าสกีมารองรับหน่วยการเรียนรู้อื่นๆ

│   ├── style.css               # สไตล์ชีตแกนหลักและระบบ Interactive UI ของข้อสอบ
│   ├── responsive.css          # ตัวปรับแต่งการแสดงผลบน Tablet และ Mobile
│   └── dark-mode.css           # ตัวควบคุมโทนสีเข้ม (Dark Mode) ของทั้งระบบ

│   └── quiz.js                 # แกนควบคุมกลไกทำข้อสอบปรนัยและทำไฮไลต์ขอบสี
                    # โฟลเดอร์เก็บสื่อมัลติมีเดียและเอกสารดาวน์โหลด
    ├── images/
    ├── docs/
    └── ppt/
