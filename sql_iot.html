ศูนย์รวมมินิเกมทักษะคำสั่งฐานข้อมูลจากบทเรียน IoT (10 ข้อ ข้อละ 10 วินาที)

เลือกหนึ่งในสี่เกมประลองไหวพริบด้านล่างเพื่อเริ่มการสปีดรันเก็บคะแนนทันที:

🛡️
Database Defender
สวมบทบาทเป็นผู้ควบคุมคำสั่ง CRUD ให้ปลอดภัย คอยจับตาเงื่อนไข WHERE ในจังหวะลบและแก้ไขข้อมูล เพื่อพยุงระบบไม่ให้เสียหาย
⛓️
CASCADE Reaction
ท้าทายระบบการผูกโครงสร้างข้อมูลและเงื่อนไข Foreign Key ตารางแม่-ตารางลูก วิเคราะห์ทิศทางและปฏิกิริยาสะสมข้อมูลตกค้าง
🕵️‍♂️
SQL Detective
ระบุรอยรั่วและคำสั่งประสงค์ร้ายที่แฮกเกอร์ฝังตัวก่อกวนระบบ Dashboard ตึกสมาร์ทเซนเซอร์ ตรวจสอบ Log ภายใน 10 วินาที!
⚡
Syntax Speedrunner
มินิเกมวิ่งเร็วเติมคำศัพท์ทางไวยากรณ์ SQL IoT จับเวลาสลับคีย์เวิร์ด ตัวแปร และเครื่องหมาย ; (Semicolon) ให้แม่นยำ
