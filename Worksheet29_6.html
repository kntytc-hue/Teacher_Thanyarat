Teacher Guideline

คู่มือการเตรียมระบบก่อนเริ่มใบงานที่ 9

สำหรับคุณครูผู้สอนวิชาระบบเครือข่ายคอมพิวเตอร์ (ปวช.2)

🛠️
1

วิธีที่ 1: ใช้คอมพิวเตอร์ของครูทำเป็น Server จำลอง

แนะนำที่สุด 👍

ถ้าห้องแล็บคอมพิวเตอร์มีระบบสาย LAN เชื่อมต่อกันอยู่แล้ว ให้คุณครูเปลี่ยนเครื่องคอมพิวเตอร์ของครูที่หน้าห้องให้กลายเป็น FTP Server ครับ

  • ✔ ดาวน์โหลดและติดตั้งโปรแกรมชื่อ FileZilla Server (ดาวน์โหลดฟรี) ลงบนเครื่องครู
  • ✔ ตั้งค่าสร้างบัญชีผู้ใช้ (User) ในโปรแกรม โดยกำหนดตามในใบงานคือ:
    Username: std_pvoc2
    Password: password2026
  • ✔ ดูหมายเลข IP Address ของเครื่องคุณครู (เช่น 192.168.1.100) เพื่อเอาไปเขียนบนกระดานให้นักเรียนพิมพ์ในคาบเรียน
2

วิธีที่ 2: ใช้เว็บเซิร์ฟเวอร์ฟรีบนอินเทอร์เน็ต (กรณีไม่อยากตั้งเซิร์ฟเวอร์เอง)

หากคอมพิวเตอร์ในห้องเรียนสามารถต่ออินเทอร์เน็ตออกข้างนอกได้ทุกเครื่อง คุณครูไม่จำเป็นต้องตั้ง Server เองครับ ให้ใช้บริการ FTP ฟรีบนอินเทอร์เน็ตแทน (เรียกว่า Online FTP Sandbox)

คุณครูสามารถไปสมัครเว็บฝากโฮสต์ฟรี (เช่น 000webhost หรือเว็บทดสอบ FTP ฟรีเช่น dlptest.com) แล้วนำค่า Host, Username, Password ที่เว็บนั้นให้มา ไปเขียนระบุบนกระดานดำให้นักเรียนทำตามแทนครับ

📋 สรุป Loadout สิ่งที่ครูต้องเขียนบนกระดานก่อนเริ่มใบงาน

เมื่อคุณครูเตรียมระบบตามข้อข้างบนเสร็จแล้ว พอถึงเวลาเรียน สิ่งแรกที่ต้องทำคือเขียนข้อมูลนี้ทิ้งไว้บนกระดานดำหน้าห้อง เพื่อให้นักเรียนเอาไปกรอกในโปรแกรมครับ:

📌 ข้อมูลสำหรับทำใบงานที่ 9 (ให้นักเรียนกรอกใน FileZilla): Port Default: 21

Host: [ใส่เลข IP เครื่องครู หรือชื่อเว็บ FTP ฟรี]

Username: std_pvoc2

Password: password2026

Port: 21

* เมื่อเตรียมข้อมูลตรงนี้พร้อมแล้ว คุณครูถึงจะสั่งให้นักเรียนเดินเครื่อง เปิดโปรแกรม FileZilla Client แล้วเริ่มทำตามขั้นตอนข้อ 1, 2, 3 ในใบงานได้เลยครับ!
